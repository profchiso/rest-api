REQUIREMENTS

To run the API, the user needs the following to perform CRUD operations

  1. Install json-server
  2. Web browser (preferably Google Chrome)
  3. POSTMAN application
  4. JSONview a google chrome extension to help format the views (optional)

How to run

open the project folder in a terminal and type json-server --watch db.json

below are the basic entry points

  1. GET /developers
  2. GET /developers/1
  3. POST /developers
  4. PUT /developers/1
  5. PATCH /developers/1
  6. DELETE /developers/1
  1. To view all developers

    Type this url http://localhost:3000/developers in POSTMAN Address bar and click on send

    Note the HTTP Method must be GET

  2. To view a developer with id

    Type this url http://localhost:3000/developers/id in POSTMAN Address bar and click on send, where the id is an integer say 1

    e.g http://localhost:3000/developers/1

    Note the http method must be GET

  3. To add a developer

    Type this url http://localhost:3000/developers in POSTMAN Address bar ,
    below the url in POSTMAN app select Body
    below the Body select x-www-form-urlencoded
    then enter the fieldnames in the KEY and the values in the VALUE

    Note the http method must be POST

  4. To update some fields say fullname, stack of a developer with a given id

    Type this url http://localhost:3000/developers/id in POSTMAN Address bar ,
    below the url in POSTMAN app select Body
    below the Body select x-www-form-urlencoded
    then enter the field name that you want to update in the KEY and the value you want it to be changed to in the VALUE
    where the id is an integer say 1

    e.g http://localhost:3000/developers/1

    Note the http method must be PATCH

  5. To update all fields of a developer with a given id

    Type this url http://localhost:3000/developers/id in POSTMAN Address bar ,
    below the url in POSTMAN app select Body
    below the Body select x-www-form-urlencoded
    then enter the all the fields in the KEY and the value you want them to be changed to in the VALUE then click send
    where the id is an integer say 1

    e.g http://localhost:3000/developers/1

    Note the http method must be PUT

  6. To delete a developer with a given id

    Type this url http://localhost:3000/developers/id in POSTMAN Address bar , then click send
    where the id is an integer say 1

    e.g http://localhost:3000/developers/1

    note the http method must be DELETE

  7. To sort the developers using any of the fields

    Type this url http://localhost:3000/developers?_sort=fieldname&_order=asc or desc in POSTMAN Address bar , then click send
    where the fieldname can be any of the field say stack

    e.g http://localhost:3000/developers?_sort=stack&_order=asc

    Note the http method must be GET

  8. You can also search a developer using any of the fields say email

    Type this url http://localhost:3000/developers?fieldname=fieldValue in POSTMAN Address bar , then click send
    where the fieldname can be any of the field say email

    e.g http://localhost:3000/developers?email=okoriechinedusunday@gmail.com

    Note the http method must be GET
